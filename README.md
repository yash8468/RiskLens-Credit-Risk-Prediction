# RiskLens – Credit Risk Prediction System

An end-to-end supervised machine learning system designed to classify loan applicants into risk categories, helping financial institutions mitigate credit risk.

### 🛠️ Technical Highlights
- **Custom Implementations:** Developed **Logistic Regression** and **K-Nearest Neighbors (KNN)** from scratch to understand core algorithmic logic.
- **Ensemble & Classic Models:** Integrated Random Forest, SVM, and Naive Bayes for performance benchmarking.
- **Full Pipeline:** Managed the complete lifecycle including EDA, data preprocessing, feature engineering, and model training.
- **Evaluation:** Conducted rigorous model comparison using Precision, Recall, F1-Score, and Accuracy.

### 🚀 Workflow
`Data Acquisition` → `Exploratory Data Analysis (EDA)` → `Feature Engineering` → `Model Training` → `Evaluation` → `Prediction`
